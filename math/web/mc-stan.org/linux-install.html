Stan: Linux Command Install

Installing C++ Development Tools

On Linux, C++ and make are often installed by default.

To see if the g++ compiler and make are installed, try

% g++ --version

and

% make --version

To install the latest version of these tools (or upgrade an older version), use

% sudo apt-get install g++

and

% sudo apt-get install make

A password will likely be required by the superuser command sudo.

Stan Source

Download the most recent stable version of

  • stan-src-1.m.p.tgz
    (m is the minor version and p the patch level)

from

  • Stan Downloads

to the directory where Stan will reside.

In the command shell, change directories to where the tarball was downloaded, <download-dir>.

% cd <download-dir>

Unpack the distribution into the subdirectory

  • <download-dir>/stan-src-1.m.p/

with

% tar -xzf stan-src-1.m.p.tgz
