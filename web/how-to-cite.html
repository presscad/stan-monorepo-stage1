Stan: How to Cite

If you refer to Stan in a paper or report, please let us know about it so we can include it in our list of citations.

Citing the Stan Software Package

  • Stan Development Team. 2012. Stan: A C++ Library for Probability and Sampling, Version 1.0.   http://mc-stan.org/.

Citing the Stan Reference Manual

  • Stan Development Team. 2012. Stan Modeling Language User's Guide and Reference Manual, Version 1.0. 
@manual{stan-manual:2012,
    author = {{Stan Development Team}},
    year = 2012,
    title = {Stan Modeling Language User's Guide and
             Reference Manual, Version 1.0},
    url = {http://mc-stan.org/}
}

Citing Stan's Sampler

  • Hoffman, Matthew D. and Andrew Gelman. In press. The No-U-Turn Sampler: Adaptively Setting Path Lengths in Hamiltonian Monte Carlo. Journal of Machine Learning Research. 
@article{hoffman-gelman:2012,
    author = {Matthew D. Hoffman and Andrew Gelman},
    year = {In Press},
    title = {The No-{U}-Turn Sampler: Adaptively Setting Path 
             Lengths in {H}amiltonian {M}onte {C}arlo},
    journal = {Journal of Machine Learning Research}
}
